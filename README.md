# Algorithm Notebook

A personal repository for competitive programming studies, algorithm notes, problem-solving techniques, and solutions from previous contests and olympiads.

This repository serves as both a learning journal and a structured knowledge base, containing implementations, explanations, observations, and solved problems gathered throughout my training journey.

## Contents

- 📚 Algorithm notes
- 🧠 Data structures
- ⚡ Competitive programming techniques
- 📝 Problem-solving strategies
- 🏆 Solutions from contests and olympiads
- 🔍 Analysis and observations
- 💻 Code implementations
- 📖 References and study materials
